# DMPUILib
Swift UI library for DMP BLE iOS application

## Components
- DMPSliderView
```swift
DMPSliderView(
    value: Binding<Int>,
    maxHeight: Int = 400,
    maxWidth: CGFloat = 150,
    sliderNumbers: Int = 15,
    tint: Color = .blue,
    cornerRadius: CGFloat = 30.0,
    padding: CGFloat = 0.5,
    clipped: Bool = true
)
```
